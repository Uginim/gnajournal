MDX는 Markdown에 JSX를 결합한 포맷입니다. 일반 Markdown처럼 글을 작성하면서도, 필요한 곳에 컴포넌트를 삽입할 수 있습니다.

## MDX의 장점

- Markdown의 간편함을 그대로 유지
- 필요한 곳에 인터랙티브 컴포넌트 삽입 가능
- Astro 컴포넌트와 완벽 호환

## 사용 예시

일반 Markdown 문법은 그대로 사용할 수 있습니다:

- **굵은 글씨**와 *기울임*
- [링크](https://astro.build/)
- 코드 블록

```astro
---
// .mdx 파일에서 Astro 컴포넌트를 import할 수 있습니다
import MyComponent from '../../components/MyComponent.astro';
---

<MyComponent />
```

MDX 파일의 확장자는 `.mdx`이며, `src/content/blog/` 디렉토리에 저장하면 자동으로 블로그 글로 인식됩니다.
